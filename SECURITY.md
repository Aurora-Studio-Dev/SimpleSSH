# 安全策略

## 支持的版本

| Version | Supported          |
| ------- | ------------------ |
| 5.1.x   | :white_check_mark: |
| 5.0.x   | :x:                |
| 4.0.x   | :white_check_mark: |
| dev 0.x | :x:                |

## 报告 Bug

前往 [GitHub](https://github.com/Aurora-Studio-Dev/SimpleSSH/issues) 提交详细的 Bug 描述，包括复现步骤、预期行为、实际行为、环境信息等。
